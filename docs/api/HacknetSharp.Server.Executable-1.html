IsPathFullyQualified(String)

Returns true if the path is fixed to a specific drive or UNC path. This method does no validation of the path (URIs will be returned as relative as a result). Returns false if the path specified is relative to the current drive or working directory.

Declaration
public static bool IsPathFullyQualified(string path)
Parameters
Type Name Description
System.String path
Returns
Type Description
System.Boolean
Remarks

Handles paths that use the alternate directory separator. It is a frequent mistake to assume that rooted paths System.IO.Path.IsPathRooted(System.String) are not relative. This isn't the case. "C:a" is drive relative- meaning that it will be resolved against the current directory for C: (rooted, but relative). "C:\a" is rooted and not relative (the current directory will not be used to modify the path).

System.ArgumentNullException

Thrown if path is null.
